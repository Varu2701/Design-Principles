# Task : Implement a Library System using the Single Responsibility Principle
## Description : 
Design a basic library management system where book-related functionality is handled by a 'BookManager' class, and user-related functionality is handled by a separate 'UserManager' class. Ensure each class has only one responsibility.
## Topics Covered : 
Single Responsibility Principle
## Objective : 
Learner will understand how to separate responsibilities into distinct classes to adhere to the Single Responsibility Principle.
